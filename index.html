Pokémon Reborn Database

What's this website for?

Basically, this website is pokemondb, but for Pokémon Reborn. It's a place where you can get information on Pokémon, moves, items, etc., where the information will be valid for Pokémon Reborn, where Reborn differs from the official Pokémon games. It also contains Reborn-specific information on encounters, trainers, etc.
