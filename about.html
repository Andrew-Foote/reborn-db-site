Can I download your data?

The data has been extracted from the files in the Pokémon Reborn installation. So in a sense, assuming you've downloaded Pokémon Reborn, you already have the data. But I wrote some Python code to collect all this data into a more manageable form, specifically a SQLite database. A copy of the database is available for download from here although I can't guarantee that it will always be up-to-date with this website as I have to remember to update it separately from the code. Another approach if you have some coding skills would be to download the code behind the site from here, and use that to build the database (it only requires a Pokémon Reborn installation to be present on your computer), although I haven't really documented the code very well.

Ideally, I'd just provide the database file from a page on this website, but unfortunately my current hosting provider (GitHub Pages) doesn't work well with such large files. (It's only about 100 megabytes, which is very small for a database, but still a bit too large for GitHub Pages.) Finding a better hosting solution is on my to-do list.
