Can I download your data?

Most of the data has been extracted from the files in the Pokémon Reborn installation. So in a sense, assuming you've downloaded Pokémon Reborn (otherwise, why are you looking at this website?), you already have the data. But I wrote some Python code to collect all this data into a more manageable form, specifically a SQLite database; you can see the code here. You can use this code on your own Pokémon Reborn installation to build the database, although I haven't really documented how it works yet (it's on my to-do list) so for now you'll just have to figure it out yourself.

Ideally, I'd just provide the database file here for you to download, but unfortunately my current hosting provider (GitHub Pages) doesn't work well with such large files. (It's only about 75 megabytes, which is very small for a database, but still a bit too large for GitHub Pages.) Finding a better hosting solution is on my to-do list.
