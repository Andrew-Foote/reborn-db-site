Shadow Force

Damage class Physical Type GHOST Target Single target
Power 120 Accuracy 100 PP 5
Priority 0 Additional effect chance 0%
Additional effect description User must wait a turn before use, but is invulnerable during that turn. Hits protected target and removes their protection.

In-game description

The user disappears, then strikes on the next turn. It strikes even if the target protects itself.
