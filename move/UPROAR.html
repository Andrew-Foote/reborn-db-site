Uproar

Damage class Special Type NORMAL Target Single opponent selected randomly
Power 90 Accuracy 100 PP 10
Priority 0 Additional effect chance 0%
Additional effect description User is locked into this move for 2 more turns. No Pokémon on the field can sleep during this time.

In-game description

The user attacks in an uproar for three turns. During that time, no Pokémon can fall asleep.
