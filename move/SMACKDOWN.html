Smack Down

Damage class Physical Type ROCK Target Single target
Power 50 Accuracy 100 PP 15
Priority 0 Additional effect chance 100%
Additional effect description Grounds the target. Hits targets using Fly, Bounce or Sky Drop, and cancels Fly or Bounce.

In-game description

The user throws a stone or a similar projectile to attack. Flying Pokémon fall to the ground when hit.
