Ominous Wind

Damage class Special Type GHOST Target Single target
Power 60 Accuracy 100 PP 5
Priority 0 Additional effect chance 10%
Additional effect description Increases every stat of the user except HP, accuracy and evasion by 1 stage.

In-game description

The user blasts the target with a gust of repulsive wind. This may also raise all the user's stats at once.
