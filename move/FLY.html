Move List

Fly

Damage class Physical Type FLYING Target Single target
Power 90 Accuracy 95 PP 15
Priority 0 Additional effect chance 0%
Additional effect description User must wait a turn before use, but is in the sky and invulnerable to most attacks during that turn.

In-game description

The user soars, then strikes the target on the next turn. It can be used to fly to any familiar location.
