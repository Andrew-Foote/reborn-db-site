Focus Punch

Damage class Physical Type FIGHTING Target Single target
Power 150 Accuracy 100 PP 20
Priority -3 Additional effect chance 0%
Additional effect description Only hits if user is not hit by a damaging move this turn.

In-game description

The user focuses their mind before launching a punch. The move fails if the user is hit before using it.
