Spirit Break

Damage class Physical Type FAIRY Target Single target
Power 75 Accuracy 100 PP 15
Priority 0 Additional effect chance 100%
Additional effect description Increases user's Special Attack by 1 stage.

In-game description

The user attacks the target with so much force that it could break the target's spirit. This also lowers the target's Sp. Atk stat.
