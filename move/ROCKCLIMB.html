Rock Climb

Damage class Physical Type NORMAL Target Single target
Power 90 Accuracy 85 PP 20
Priority 0 Additional effect chance 20%
Additional effect description Inflicts confusion.

In-game description

The user attacks the target by smashing into it with incredible force. This may also confuse the target. It can also be used to scale rocky walls.
