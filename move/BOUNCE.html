Bounce

Damage class Physical Type FLYING Target Single target
Power 85 Accuracy 85 PP 5
Priority 0 Additional effect chance 30%
Additional effect description User must wait a turn before use, but is in the sky and invulnerable to most attacks during that turn. Inflicts paralysis.

In-game description

The user bounces up high, then drops on the second turn. This may also paralyze the target.
