Snatch

Damage class Status Type DARK Target Self
Power --- Accuracy --- PP 10
Priority 4 Additional effect chance 0%
Additional effect description For the duration of the current turn, negates the effect of any positive stat-changing moves used by any Pokémon on the field, and applies their effects to the user instead.

In-game description

The user steals the effects of any attempts to use a healing or stat-changing move.
