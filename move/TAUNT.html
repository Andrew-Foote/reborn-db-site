Taunt

Damage class Status Type DARK Target Single target
Power --- Accuracy 100 PP 20
Priority 0 Additional effect chance 0%
Additional effect description Adds a temporary status condition to the target which prevents them from using non-damaging moves.

In-game description

The target is taunted into a rage that allows it to use only attack moves for three turns.
